# Interactive mode

- **`docker run -it {$image}`**
    - `-i` : Interactive mode로 container에 input값을 입력할 수 있게 됨. (Keep STDIN open even if not attached)
    - `-t` : 터미널 생성 (Allocate psudo-TTY )

- **`docker start -a -i {$container_name}`**
    - `-a` : Attach mode로 실행(출력 값 볼 수 있게됨). 이때, `-a` 만으로도 Input 값을 입력할 수 있으나, 단 한번만 입력 가능
    - `-i` : Interactive mode로 실행 (`-t`는 처음 image 파일에서 container를 생성했을 때 입력했으면 더 이상 입력할 필요 X)
